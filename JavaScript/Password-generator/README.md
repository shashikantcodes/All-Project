# 🔐 Password Generator

## 📄 Description:

A simple tool to generate random secure passwords using JavaScript.

## 🚀 Features:

- Choose Length
- Include Symbols, Numbers, Uppercase, Lowercase
- Copy to Clipboard

## 🛠 Tech Stack:

HTML, CSS, JavaScript

## 📂 How to Run:

Open index.html in browser and use the generator UI.
