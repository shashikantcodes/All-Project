# 🎲 Random GIF Generator

## 📄 Description:
A fun React app that fetches and displays random GIFs using Giphy API.

## 🚀 Features:
- Random GIFs on Click
- Search Custom GIFs
- Giphy API Integration

## 🛠 Tech Stack:
React JS, Tailwind CSS, Giphy API

## 📂 How to Run:
bash
npm install
npm start
